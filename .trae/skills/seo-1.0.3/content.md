# 内容 SEO

## 搜索意图
将内容类型与查询意图匹配 — 不匹配 = 无排名：
- **信息型**："如何"、"是什么" → 指南、教程、解释
- **导航型**：品牌 + 产品 → 主页、产品页面
- **交易型**："购买"、"价格"、"折扣" → 带 CTA 的产品/服务页面
- **商业调查型**："最佳"、"vs"、"评论" → 比较、评论

通过搜索检查意图：第 1 页上排名的是什么？匹配该格式。

## E-E-A-T
对于 YMYL（你的钱你的生活）主题——健康、金融、法律、安全：
- **经验**：主题的第一手经验（评论、案例研究）
- **专业知识**：作者凭证、带简介的署名
- **权威性**：网站声誉、来自权威来源的引用
- **可信度**：联系信息、隐私政策、安全网站、准确内容

E-E-A-T 信号：
- 带凭证和照片的作者简介
- 带公司历史的关于页面
- 到权威来源的外部链接
- 内容上的更新日期
- 为声明引用来源

## 内容质量
- 薄内容惩罚：<300 字且无独特价值 → noindex 或扩展
- 重复内容：Google 选择一个版本，其他被过滤 → 使用 canonical
- 内容深度：全面覆盖主题，回答相关问题
- 独特价值：你的内容提供竞争对手没有的什么？
- 用户参与度：页面停留时间短和跳出率高信号差内容

## 新鲜度
- 查询值得新鲜内容（QDF）：时间敏感主题需要最近内容
- "2025 年最佳 X" 查询：带日期的内容排名，但必须实际更新
- 更新日期：只改日期不改内容 = 垃圾信息
- 常青内容：URL 中没有日期，定期更新内容
- 新闻：需要快速索引，新闻网站使用 Google News 站点地图

## 内容结构
- 在前 100 个字中回答查询——精选片段潜力
- 长内容的目录——改善 UX 并可能显示为站点链接
- 底部的 FAQ 部分——捕获"人们还问"查询
- 项目符号和编号列表——精选片段的首选
- 短段落（2-3 句话）——更容易阅读，尤其是在移动端
