# 设置 — SEO

## 首次使用

此技能无需设置即可立即工作。

## 工作区集成

为了跨会话持久跟踪，创建工作区：

```
mkdir -p ~/seo/audits ~/seo/content
```

在你的主记忆文件中注册：

```markdown
## SEO 工作区
位置：~/seo/
- 审核报告：~/seo/audits/
- 内容草稿：~/seo/content/
```

## 可选配置

在 `~/seo/memory.md` 中跟踪特定网站：

```markdown
# SEO 记忆

## 网站
- example.com — 主要项目，e-commerce
- blog.example.com — 内容中心

## 关键词跟踪
| 关键词 | 当前排名 | 目标 | 上次检查 |
|-------|---------|------|---------|
| best widgets | 12 | 3 | 2025-01-15 |
```

## 无需设置

该技能无需任何配置即可提供完整的 SEO 指导。对于想要跟踪多个网站或审核历史的用户，工作区设置是可选的。
