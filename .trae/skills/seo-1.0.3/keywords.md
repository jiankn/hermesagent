# 关键词研究与竞争分析

## 关键词研究流程

1. **种子关键词** — 从主题的明显术语开始
2. **扩展** — 找到相关术语、问题、长尾变体
3. **分析** — 检查搜索量、难度、意图
4. **确定优先级** — 平衡量、难度和商业价值
5. **映射** — 为页面分配关键词（每页一个主要关键词）

## 关键词指标

| 指标 | 含义 | 操作 |
|------|------|------|
| 搜索量 | 每月搜索次数 | 越高 = 越多潜在流量 |
| 关键词难度 | 竞争水平 | 越高 = 越难排名 |
| CPC | 每次点击广告成本 | 越高 = 越多商业价值 |
| SERP 功能 | 片段、地图、图像 | 影响点击率潜力 |

## 关键词类型

**按意图：**
- 信息型："如何"、"是什么"、"指南"
- 导航型：品牌名称、特定网站
- 交易型："购买"、"价格"、"折扣"、"最佳"
- 商业型："vs"、"评论"、"替代"

**按长度：**
- 头部术语：1-2 个字，高量，高竞争
- 长尾：3+ 个字，低量，更容易排名，更高转化

## 竞争分析

### 分析什么

1. **谁排名** — 目标关键词的前 10 名
2. **他们覆盖什么** — 主题、子主题、深度
3. **他们如何结构** — 格式、标题、媒体
4. **他们链接到哪里** — 内部和外部链接
5. **他们为什么排名** — 权威性、内容质量、技术

### 差距分析

找到竞争对手错过的机会：
- 他们不覆盖的主题
- 他们不回答的问题
- 他们不使用的格式（视频、工具、计算器）
- 他们缺乏的深度
