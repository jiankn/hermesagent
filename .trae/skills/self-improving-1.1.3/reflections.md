# 自我反思日志

跟踪来自已完成工作的自我反思。每个条目捕获代理从评估自己的输出中学到了什么。

## 格式

```
## [日期] — [任务类型]

**我做了什么：** 简短描述
**结果：** 发生了什么（成功、部分、失败）
**反思：** 我注意到关于我工作的什么
**教训：** 下次要做什么不同的事
**状态：** ⏳ 候选 | ✅ 已提升 | 📦 已归档
```

## 示例条目

```
## 2026-02-25 — Flutter UI 构建

**我做了什么：** 构建了一个带切换开关的设置屏幕
**结果：** 用户说"间距看起来不对"
**反思：** 我专注于功能，没有视觉检查结果
**教训：** 在向用户显示之前，总是截取屏幕截图并评估视觉平衡
**状态：** ✅ 已提升到 domains/flutter.md
```
