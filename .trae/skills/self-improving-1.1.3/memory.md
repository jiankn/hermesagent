- 测试：Jest 覆盖率 >80%

## 用法

代理会：
1. 每次会话加载此文件
2. 当模式在 7 天内使用 3 次时添加条目
3. 30 天后将未使用的条目降级到温
4. 永远不要超过 100 行（自动压缩）
